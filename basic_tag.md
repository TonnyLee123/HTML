## 標題及段落
```
1. <h1>...</h1>
- Header
- h1 ~ h2

2. <p>...</p>

3. <br/>
- Single tag
- Create new line

4. <hr/>
- Horizontal line
```

## 字型編輯
- 粗體
- 斜體
- 字體加大
- 字體縮小
```
1. <b>...</b>
- Bold Text

2. <i>...</i>
- Italicize

3. <big>...</big>
- 放大字體

4. <small>...</small>
- 縮小字體

5. <sub>...</sub>
- Subscript
- 縮小字體
- <p>H<sub>2</sub>O</p>

6. <sup>...</sup>
- Superscript
- 指數
- <p>10<sup>2</sup></p>
```
