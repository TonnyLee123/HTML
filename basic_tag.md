## 標題及段落
```
1. <h1>...</h1>
- Header
- h1 ~ h2

2. <p>...</p>
- Paragraph

3. <br/>
- Single tag
- Create new line

4. <hr/>
- Horizontal line
```

## 字型編輯
```
1. <b>...</b>
- Bold Text

2. <i>...</i>
- Italicize

3. <big>...</big>
- 放大字體

4. <small>...</small>
- 縮小字體

5. <sub>...</sub>
- Subscript
- 縮小字體
- <p>H<sub>2</sub>O</p>

6. <sup>...</sup>
- Superscript
- 指數
- <p>10<sup>2</sup></p>
```

# Style & Color
- Change text's color
```html
<p style = 'color : blue'>You can style your HTML.</p>
``` 
- Change background's color
```html
<body style = 'background-color : blue'> ... </body>
```
