# 1. Style Attribute
- used to add styles to an element, such as color, font, size, and more.
```
style="property:value;"
The property is a CSS property. The value is a CSS value.
```

## 2. 常見的 Style Attribute 
- Text's color
```html
<p style = 'color : blue;'>You can style your HTML.</p>
``` 
- Background's color
```html
<body style = 'background-color : blue;'> ... </body>
```
- Fonts
```html
<h1 style="font-family:verdana;">...</h1>
```
- Text Size
```html
<h1 style="font-size:300%;">...</h1>
```

- Text Alignment
```html
<p style="text-align:center;">...</p>
```

# Text Formatting
```
<b> - Bold text
<strong> - Important text (Blod text)
<i> - Italic text
<mark> - Marked text
<small> - Smaller text
<del> - Deleted text
<ins> - underline text
<sub> - Subscript text
<sup> - Superscript text
```
## 範例
```html
<p>H<sub>2</sub>O</p>
<p>H<sup>2</sup>O</p>
```
